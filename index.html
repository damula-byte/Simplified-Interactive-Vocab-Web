How to Deploy & Secure Database ▼

1. Setup Firebase: Go to firebase.google.com, create a project, and enable "Realtime Database".

2. Security Rules: In the Realtime Database "Rules" tab, paste this:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

(Note: These are open rules for easy setup. For production, restrict writes to authenticated users).

3. Host anywhere: Upload the whole project folder (index.html, css/, and js/) to GitHub Pages, Netlify, or any static hosting. No server-side code needed - just keep the folder structure intact.
